This Website

Github Repo

This website is an attempt at showing off my ability to create a front end applicaiton using vanilla javascript, or more accuratly Typescript. There is a little bit of reinventing the wheel, but I wanted as much of this website as possible to be my own code. The only things that aren't are the functions used to access Google's firestore database and some of the code that is generated by webpack when it is compiling my typescript.

The website works with a class that disables the browsers main functions when clicking links in the navbar and loading content from a router. The router can either hold static content or load content from another HTML file. The router also has the ability to load javascript into a script tag and execute it after the HTML has been inserted into the page.

The last thing that I added to the website was the ability to template an html file. I chose two brackets to be used to insert a variable into the page. Ex: 

\{\{ variable \}\}
With this method I could introduce functions that could call other template pages and loop over arrays, but I still wasn't satisfied. I ended up adding the ability to execute javascript code within different dilimiters opting for brackets and percent sign. Ex: 
\{\% javascript code \%\}
This allowed me to include things like testing for a missing variable or formating a date a certain way, or looping over an array in a more clean fassion.

The functions that I have included into the rendering are:

  • For Each - That inclueded a callback that was called for each item within an array, and a callback for if the array was empty.
  • Include - That took a string for a filename of the template to include in that location and an args variable
  • Formate Date - that took a format string and formated a date to look nice.

Phasmophobia Helper

Find it Here | Github Repo

Phasmophobia is a horror puzzle game where you enter a haunted house and attept to discover the type of ghost haunting the house. There are varying levels of difficulty where not all the evidence needed to identify the ghost are presented instead rellying on "secondary evidence". This app will remove ghosts from the list based on the amount of evidence that get provided and help narrow down the ghost you are trying to discover.

I created this website using Typescript to create classes that interact with the dom, abstracting the logic of the website from the dom manipulation. Webpack is used to bundle it all together, and is hosted, like most of my projects on firebase.
